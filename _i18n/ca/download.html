Requisits

Requisits del sistema

L'OmegaT funcionarà en qualsevol sistema en què estigui instal·lada o es pugui instal·lar una versió adequada del JRE (Java Runtime Environment). El JRE ara s'inclou amb l'OmegaT i no cal obtenir-lo per separat. L'OmegaT s'ha instal·lat correctament en el Windows (totes les versions a partir del Windows 98), Macintosh OS X i Linux.

Els fitxers en format XML del Microsoft Office es poden traduir directament en l'OmegaT. Els fitxers en formats antics del MS Word, Excel i Powerpoint es poden traduir després de convertir-los als formats actuals del Microsoft Office equivalents, o bé al format Office Open XML, conversió que es pot realitzar mitjançant l'OpenOffice.org o el LibreOffice (també lliures). Per a obtenir l'OpenOffice.org o el LibreOffice, seguiu l'enllaç del botó «Programari de tercers» a l'esquerra.

Requisits de l'usuari

L'OmegaT no és especialment difícil d'instal·lar o d'utilitzar, però requereix disposició a llegir i seguir les instruccions. Si teniu una aversió natural a la lectura d'instruccions i la vostra actitud amb el programari nou es basa a fer clic en cada botó que veieu fins que passa alguna cosa semblant a l'efecte desitjat, aleshores l'OmegaT probablement no és adequat per a vós.

Versions obsoletes

Encara podeu trobar versions anteriors i desfasades de l'OmegaT.

Les versions obsoletes són versions antigues que han quedat desfasades. Per a la majoria d'usuaris, no són d'interès. Ara bé, alguns usuaris poden patir restriccions (imposades pels seus sistemes operatius o versió del Java) que poden provocar que prefereixin una versió anterior.

Amb el JRE o sense

Per a executar-se, l'OmegaT requereix que tingueu el Java Runtime Environment (JRE) instal·lat a l'ordinador.

Si esteu segur que ja teniu instal·lada una versió adequada del JRE en el vostre ordinador, podeu baixar una versió de l'OmegaT sense el JRE. La baixada serà més ràpida i la instal·lació requerirà menys espai en el disc dur. En aquest cas, no seleccioneu la versió per defecte; seleccioneu una versió diferent (p. ex., estàndard o més recent) i seleccioneu una versió «sense el Java».

Si instal·leu la versió de l'OmegaT amb el JRE encara que ja tingueu instal·lat el JRE a l'ordinador, el JRE de l'OmegaT s'instal·larà per separat i no entrarà en conflicte. Per tant, sempre és segur instal·lar la versió de l'OmegaT amb el JRE. Si no esteu segur de si teniu instal·lada o no una versió adequada del JRE, seleccioneu una versió amb el JRE (o la versió per defecte).

Usuaris del Linux: la majoria de distribucions del Linux proporcionen el Java Runtime Environment, però algunes implementacions de codi obert no funcionen correctament amb l'OmegaT (encara que no sigui aparent immediatament). Si fa relativament poc que utilitzeu el Linux, us aconsellem que seleccioneu una versió de l'OmegaT que inclogui el JRE. És possible que els usuaris del Linux amb més experiència prefereixin instal·lar el JRE d'Oracle (anteriorment, Sun) per a tot el sistema.

Versió validada o versió sense signar

El macOS intenta incrementar la seguretat dificultant l'execució de programes sense verificar.

L'OmegaT per al macOS està disponible en versions validades i sense signar; funcionalment són idèntiques, però la versió validada és preferible perquè el sistema reconeix que a) l'ha publicat un desenvolupador conegut i b) l'ha analitzat Apple.

Pot haver-hi un retard en la publicació de la versió validada. Els usuaris amb pressa poden utilitzar la versió sense signar en el seu lloc; si teniu problemes per a executar l'aplicació, consulteu com obrir una aplicació d'un desenvolupador no identificat.

Durant la instal·lació, segons el vostre sistema operatiu, és possible que rebeu diversos avisos de seguretat.
El certificat està signat pel desenvolupador principal de l'OmegaT.

Els permisos que proporcioneu a aquesta versió (que es poden descriure com a «accés sense restriccions a l'ordinador») són idèntics als permisos que assigneu a la versió local; és a dir, permeten accedir al disc dur de l'ordinador.

Informació de privadesa: l'OmegaT amb el Java Web Start no desa cap informació en els servidors web de l'OmegaT. L'aplicació s'executa en el vostre ordinador. Els documents i les memòries de traducció romanen a l'ordinador i el projecte de l'OmegaT no tindrà accés a la vostra feina o informació.
