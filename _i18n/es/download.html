OmegaT can be installed on any operating system capable of running version 11 of the Java runtime environment.

Seleccione si desea descargar OmegaT con Java Runtime Environment (JRE) incluido o sin él. Si ya tiene un JRE adecuado instalado en su sistema Windows, seleccione "sin JRE". Si no lo tiene o no lo sabe con seguridad, seleccione "con JRE".

Normalmente, OmegaT funciona en cualquier sistema Linux en el que se pueda instalar una versión adecuada de Java Runtime Environment (JRE). OmegaT currently requires version 11 of the JRE or later. Para comprobar qué versión de JRE, si hay alguna, está instalada en su sistema Linux, abra la consola (ventana de línea de comandos) y escriba:

$ java -version

Si no sabe con seguridad si ya hay una versión adecuada de JRE instalada en su sistema, puede descargar e instalar una versión de OmegaT con JRE. Esta versión de JRE solo se utilizará de manera local para OmegaT; no entrará en conflicto, anulará ni desinstalará las otras instalaciones de JRE y, por lo tanto, su uso es seguro.

La versión "con pilas incluidas" de OmegaT, con su propio Java Runtime Environment. Esta versión tardará más en descargarse y necesitará más espacio en su disco duro, pero puede estar seguro de que funcionará. En caso de duda, seleccione esta opción.

Versiones anteriores

Aún se pueden encontrar versiones más antiguas que se han sustituido de OmegaT.

Las versiones anteriores son versiones más antiguas que se han sustituido. Para la mayoría de usuarios, carecen de interés. Sin embargo, algunos usuarios pueden encontrar restricciones (impuestas por sus sistemas operativos o versión de Java) que pueden hacer que prefieran una versión más antigua.

Con/sin JRE

Es necesario que tenga Java Runtime Environment (JRE) instalado en su equipo para que OmegaT se ejecute.

Si sabe con seguridad que ya tiene una versión adecuada de JRE instalada en su equipo, puede descargar una versión de OmegaT sin JRE. La descarga será más rápida y la instalación necesitará menos espacio en el disco duro. En ese caso, no seleccione la versión predeterminada; seleccione una versión diferente (p. ej., la estándar o la última) y seleccione después una versión que no incluya Java.

Si instala la versión de OmegaT con JRE aunque ya tenga JRE instalado en su equipo, el JRE de OmegaT se instalará por separado y no entrará en conflicto. Por lo tanto, siempre es seguro instalar la versión de OmegaT con JRE. Si no sabe con seguridad si tiene o no una versión adecuada de JRE, seleccione una versión con JRE (o la versión predeterminada).

Usuarios de Linux: la mayoría de distribuciones de Linux cuentan con Java Runtime Environment, pero algunas implementaciones de código abierto no funcionan bien con OmegaT (aunque no lo parezca en un primer momento). Si es relativamente nuevo en Linux, le recomendamos que seleccione una versión de OmegaT con JRE incluido. Los usuarios de Linux más experimentados pueden preferir instalar JRE de Oracle (antes Sun) a nivel de sistema.

Autentificada vs. Sin firmar

macOS intenta aumentar la seguridad dificultando la ejecución de programas sin verificar.

OmegaT para macOS está disponible en versión autentificada y versión sin firmar. Cuentan con las mismas funciones, pero es preferible utilizar la versión autentificada porque el sistema reconoce que la publicó un desarrollador conocido y está verificada por Apple.

Es posible que se tarde un tiempo en publicar la versión autentificada. Los usuarios que tengan prisa pueden utilizar en su lugar la versión sin firmar. Si tiene dificultades para iniciar la aplicación, consulte cómo abrir una aplicación de un desarrollador no identificado.
