Requisiti

Requisiti di sistema

OmegaT funzionerà su qualsiasi sistema su cui è stato, o può essere, installato il corretto Java Runtime Environment (JRE). JRE è ora fornito con OmegaT e non deve essere ottenuto separatamente. OmegaT è stato installato con successo in Windows (dalla versione 98 in poi), Macintosh OS X e Linux.

I file in formato Microsoft Office XML possono essere tradotti direttamente in OmegaT. I file nei vecchi formati di MS Word, Excel e Powerpoint possono essere tradotti dopo la conversione negli equivalenti attuali formati Microsoft Office o, in alternativa, nel formato Office Open XML, tramite OpenOffice o LibreOffice (gratuiti). Per scaricare OpenOffice o LibreOffice, seguire il collegamento dal pulsante a sinistra "Software di terze parti".

Requisiti utente

OmegaT non è particolarmente difficile da installare o usare, ma richiede la volontà di leggere e seguire le istruzioni. Se si ha una naturale avversione per la lettura delle istruzioni e il proprio approccio al nuovo software è fare clic su ogni pulsante che si vede fino a che "qualcosa di simile" all'effetto desiderato si verifica, allora OmegaT probabilmente non è adatto a voi.

Vecchie versioni

È possibile ancora trovare le versioni più vecchie, superate di OmegaT.

Sono versioni più vecchie e superate. Per la maggior parte degli utenti esse non sono di alcun interesse. Tuttavia, alcuni utenti potrebbero dover affrontare restrizioni (imposte dal loro sistema operativo o dalla loro versione Java) che li inducono a preferire una versione precedente.

Con o senza JRE

Per essere eseguito, OmegaT richiede l'installazione nel proprio computer di Java Runtime Environment (JRE).

Se si è sicuri che una versione adatta di JRE è già installata sul proprio computer, è possibile scaricare una versione di OmegaT senza JRE. Lo scaricamento sarà più veloce e l'installazione richiederà meno spazio sul disco rigido. In questo caso, non selezionare la versione predefinita; selezionarne una diversa (ad esempio, standard o più recente), quindi selezionare una versione contrassegnata "Java not included".

Se si installa la versione di OmegaT con JRE anche se JRE è già installato nel computer, il JRE di OmegaT verrà installato separatamente e non entrerà in conflitto con esso. Pertanto è sempre sicuro installare la versione di OmegaT con JRE. Se non si è sicuri se o non si possiede una versione adatta di JRE, selezionare una versione con JRE (o quella predefinita).

Utenti di Mac OS X: JRE è installato su Mac OS X per impostazione predefinita, pertanto non esiste una versione per Mac OS X con JRE.

Utenti di Linux: la maggior parte delle distribuzioni di Linux è distribuita con Java Runtime Environment, ma alcune implementazioni open source non funzionano correttamente con OmegaT (anche se questo potrebbe non risultare immediatamente evidente). Se si è nuovi di Linux, consigliamo di selezionare una versione di OmegaT con JRE incluso. Gli utenti Linux più esperti preferiscono installare il JRE di Oracle (ex Sun) a livello di sistema.

Firmato/Non firmato

Mac OS X ha una funzionalità chiamata Gatekeeper che aumenta la sicurezza rendendo difficile l'esecuzione di programmi non firmati. OmegaT per Mac OS X è disponibile nelle due versioni, firmata e non firmata; le loro funzionalità sono le stesse ma è preferita dal sistema la versione firmata, in quando questa viene riconosciuta come pubblicata da uno sviluppatore conosciuto.

Ci potrebbero essere dei ritardi nel rilascio di versioni firmate. Gli utenti che non intendono aspettare possono utilizzare la versione non firmata; se si hanno problemi ad avviare l'applicazione, si veda come aprire un'applicazione di uno sviluppatore non identificato.

La versione Web Start offre un metodo alternativo di installazione di OmegaT sul proprio computer.

Con questa versione, i futuri aggiornamenti verranno installati automaticamente.

Durante l'installazione, a seconda del sistema operativo, è possibile ricevere vari avvisi di protezione.
Il certificato è firmato da PnS Concept Sarl, un'azienda registrata in Francia.

I permessi che si concedono a questa versione (che potrebbero essere descritti come "accesso illimitato al computer") sono gli stessi che si attribuiscono alla versione locale (per esempio, essi consentono l'accesso al disco rigido del proprio computer).

Per installare la versione Web Start, bisogna avere installato nel proprio computer Java in versione 1.3 o successiva.

Informativa sulla riservatezza: OmegaT Java Web Start non salva le informazioni dell'utente sui server web di OmegaT. L'applicazione viene eseguita sulla macchina locale. I documenti e le memorie di traduzione restano nel computer dell'utente e il progetto OmegaT non avrà accesso ai lavori o alle informazioni dell'utente.

La versione /trunk (alfa) riflette lo stato di sviluppo più recente. È consigliabile soltanto per sviluppatori esperti e collaudatori. Può essere ottenuta anche utilizzando il comando di subversion:

svn checkout svn://svn.code.sf.net/p/omegat/svn/trunk omegat-svn

Sono disponibili compilazioni periodiche su Sourceforge.

I pacchetti di codice sorgente non sono richiesti dagli utenti e sono destinati solo agli sviluppatori. Si possono trovare nelle cartelle delle corrispondenti versioni compilate.
